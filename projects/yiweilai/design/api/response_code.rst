Normal response codes::

   GET/PUT  Server Success                (200)
   POST     Resource Created　             (201)
   DELETE   No content from server　       (204)

Error response codes::
   
   Bad Request                   (400)
   Unauthorized                  (401)
   Forbidden                     (403)
   Method Not Allowed            (405)
   Service Unavailable           (503)
   Not Found                     (404)
